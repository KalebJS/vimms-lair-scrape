# TUI Game Scraper

A terminal-based application for scraping game metadata and downloading game files from Vimm's Lair.

## Purpose
Provides a modern TUI (Textual User Interface) to browse, scrape, and download retro game ROMs with queue management and progress tracking.

## Key Features
- Multi-letter category scraping with progress tracking
- Download queue with pause/resume/cancel support
- Concurrent downloads with configurable limits
- Automatic zip extraction after download
- Persistent JSON configuration
- Structured logging to files

## Ethical Guidelines
- Respects site rate limits via configurable request delays
- Does not bypass download-per-client limits
- Designed for personal/educational use only
